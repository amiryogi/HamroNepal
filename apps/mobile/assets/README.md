# HamroNepal Mobile Assets

Place the following image files in this directory:

- `icon.png` - App icon (1024x1024 px)
- `splash-icon.png` - Splash screen icon (288x288 px recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

## Design Guidelines

- Use the HamroNepal red (#dc2626) as the primary color
- Include the "हाम्रो नेपाल" text or logo
- Keep icons simple and recognizable
